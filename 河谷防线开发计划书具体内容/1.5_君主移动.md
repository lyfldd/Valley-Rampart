# 1.5 君主移动

### 优先级：🔴 最高

### 脚本和资源

| 文件 | 路径 | 说明 |
|------|------|------|
| `RulerController.cs` | `Assets/_Game/Systems/Ruler/` | 君主移动 + 输入处理 |
| `ruler_placeholder.png` | `Assets/_Game/Art/Sprites/` | 临时占位精灵 |

### 核心逻辑

```
RulerController : MonoBehaviour
  组件依赖：SpriteRenderer, Rigidbody2D

  字段：
    float moveSpeed = 5f
    float runSpeed = 8f
    bool isRunning = false（Shift 切换）

  Update() {
    读取 Horizontal 输入（A/D 或 ←/→）
    检测 Shift 按下 → isRunning = !isRunning
    float speed = isRunning ? runSpeed : moveSpeed
    rb.velocity = new Vector2(input * speed, rb.velocity.y)

    翻转 Sprite：
      if input > 0 → spriteRenderer.flipX = false（面朝右）
      if input < 0 → spriteRenderer.flipX = true（面朝左）
  }
```

### Rigidbody2D 设置

| 属性 | 值 |
|------|-----|
| Body Type | Dynamic |
| Gravity Scale | 0（横版不坠） |
| Constraints | Freeze Rotation Z ✅, Freeze Position Y ✅ |
| Collision Detection | Continuous |

### 实现步骤

1. 场景中创建空 GameObject "Ruler"
2. 添加 SpriteRenderer + Rigidbody2D + RulerController
3. 拖入临时占位 Sprite
4. 测试：运行后 A/D 左右移动，Shift 切换速度，面朝方向正确翻转

### 验证标准
- [ ] A/D 或 ←/→ 输入，君主左右移动
- [ ] Shift 切换奔跑/慢走，速度有明显差异
- [ ] 面朝方向随移动方向翻转
- [ ] 松开按键后立即停止（无滑行）
