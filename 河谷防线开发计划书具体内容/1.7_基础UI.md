# 1.7 基础 UI

### 优先级：🟡 中（依赖 1.3 EventBus）

### 脚本

| 文件 | 路径 | 说明 |
|------|------|------|
| `ResourcePanel.cs` | `Assets/_Game/UI/` | 顶部资源条 |
| `TimeIndicator.cs` | `Assets/_Game/UI/` | 日夜/时间指示器 |

### UI 布局（Canvas → Screen Space - Overlay）

```
Canvas (Canvas Scaler: Scale With Screen Size, 1920×1080)
└── TopBar (Panel, 锚定顶部, 拉伸全宽, 高度 ~60px)
    ├── ResourcePanel
    │   ├── WoodIcon + WoodText    (🪵 木: 0)
    │   ├── StoneIcon + StoneText  (🪨 石: 0)
    │   ├── GoldIcon + GoldText    (💰 金: 0)
    │   └── FoodIcon + FoodText    (🌾 粮: 0)
    └── TimeIndicator
        └── TimeText               ("第 1 天 · 白天")
```

### ResourcePanel 核心逻辑

```
ResourcePanel : MonoBehaviour
  字段：
    Text woodText, stoneText, goldText, foodText

  Start() {
    EventBus.Subscribe<ResourceChangedEvent>(OnResourceChanged)
  }
  OnDestroy() {
    EventBus.Unsubscribe<ResourceChangedEvent>(OnResourceChanged)
  }

  void OnResourceChanged(ResourceChangedEvent e) {
    woodText.text  = $"木: {e.wood}"
    stoneText.text = $"石: {e.stone}"
    goldText.text  = $"金: {e.gold}"
    foodText.text  = $"粮: {e.food}"
  }
```

### TimeIndicator 核心逻辑

```
TimeIndicator : MonoBehaviour
  字段：
    Text timeText

  Start() {
    EventBus.Subscribe<DayNightChangedEvent>(OnDayNightChanged)
  }
  OnDestroy() {
    EventBus.Unsubscribe<DayNightChangedEvent>(OnDayNightChanged)
  }

  void OnDayNightChanged(DayNightChangedEvent e) {
    timeText.text = e.isNight ? "🌙 夜晚" : "☀ 白天"
  }
```

### 实现要点

1. Canvas Scaler = Scale With Screen Size，Reference Resolution = 1920×1080
2. 四个资源图标先用 TextMeshPro 文字/emoji 占位，后期替换为 Sprite
3. 资源初始值硬编码显示 0，等 ResourceManager 实装后自动更新
4. `OnDestroy` 中必须 Unsubscribe，防止场景切换后残留回调

### 验证标准
- [ ] 顶部资源条显示四资源（木/石/金/粮），数值为 0
- [ ] 时间指示器显示"第 1 天 · 白天"
- [ ] 手动 Publish ResourceChangedEvent，UI 能实时更新
- [ ] 运行时 UI 在不同分辨率（16:9）布局正常
