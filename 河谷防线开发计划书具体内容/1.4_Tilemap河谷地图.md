# 1.4 Tilemap 河谷地图

### 优先级：最高

---

## 设计理念

横版 2D 游戏，Y 轴分三区：

```
Y轴 ┌─────────────────────────────┐
    │  Layer 0+1  背景地景         │ ← 天空 + 地形对应远景
    │             河流起点→终点     │
    ├─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┤
    │  Layer 2+5  游戏内容层        │ ← 地表线 = 玩家行走面
    │             玩家/建筑/单位     │
    │══════ 地表线 ════════════════│
    ├─────────────────────────────┤
    │  Layer 3    材质剖面层        │ ← 纯视觉贴图，不可交互
    │             数据驱动           │
    └─────────────────────────────┘
```

- **上区**：地形对应背景图（远山/森林/海洋），河流起点终点在这层绘制
- **中区**：地表线 + 玩家行走区，建筑/树木从地表向上延伸
- **下区**：地下剖面，贴图根据地表类型 + 等级决定，仅用于视觉展示和资源判定

## 地形类型总表

### 独立地形

| 地形 | 产出资源 | 分级维度 | 等级范围 | 河流影响 |
|------|:--------:|----------|:--------:|----------|
| 森林 | 木材 | 规模 | 1-3（稀疏/中等/茂密） | 可建船厂、船只运输 |
| 山地 | 石材 | 岩层 | 1-3（薄/中/厚） | 可建船厂、船只运输 |
| 矿山 | 石材+矿石 | 矿石品位 | 1-3（贫瘠/普通/富集） | 可建船厂、船只运输 |
| 沙地 | 无 | 不分级 | — | 无 |
| 雪地 | 无 | 季节驱动 | 0-3（深冬→微雪→初融→无雪） | 冬季冻结、河流不可用 |
| 山壁 | 无 | — | — | 不可通行边界 |
| 河流 | — | — | — | 自身即河流，水面 tile |

> 矿山是山地的子集：产石材也产矿石，`isMine = true`。

### 开放地面（等级驱动视觉和功能）

| 等级 | 名称 | 产出 | 条件 |
|:----:|------|:----:|------|
| 0 | 荒地 | 无 | — |
| 1 | 草地 | 无 | — |
| 2 | 平原 | 无（无河流时） | — |
| 3 | 肥沃 | 粮食（基础产出） | 平原 + hasRiver |
| 4 | 丰饶 | 粮食（2x 产出） | 平原 + hasRiver |

> 同一 tile 类型，等级越高贴图越丰富。0级干裂泥土 → 4级深色沃土。

## 河流系统

河流双重身份：
1. **独立 terrain tile**：Surface 层画水面 tile，TerrainProp 层显示"水面 + 河床"剖面
2. **相邻地形属性**：流经的 tile 标记 `hasRiver = true`

河流效果：
- 平原地形等级≥3 + hasRiver → 解锁粮食产出
- 任何有 hasRiver 的地形 → 可建船厂
- 河流 tile 上支持船只快速运输（移速加成）
- 雪地冬季等级冻结时 → hasRiver 效果暂时失效

河流背景：TerrainBg 层绘制河流从高山/平原 → 流经河谷 → 汇入左右侧海洋。

## 场景结构（6 层）

```
Main.unity
└── Grid (Grid 组件, Cell Size: 1,1,0)
    ├── SkyBg         (TilemapRenderer, Order: -2)
    ├── TerrainBg     (TilemapRenderer, Order: -1)
    ├── Surface       (TilemapRenderer, Order: 0)
    ├── TerrainProp   (TilemapRenderer, Order: 1)   ← 地表下方 N 行
    ├── Water         (TilemapRenderer, Order: 2)   ← 河流水面覆盖
    └── Decoration    (独立 SpriteRenderer 管理)     ← 建筑/树木/单位
```

> Building 层取消。建筑改为独立 GameObject + SpriteRenderer，由 BuildController 管理位置和层级。

## 数据模型

```csharp
// ===== 枚举 =====
public enum SurfaceType
{
    Forest,      // 森林
    Mountain,    // 山地
    Sand,        // 沙地
    Snow,        // 雪地（等级由 SeasonManager 驱动）
    Cliff,       // 山壁（不可通行边界）
    River,       // 河流（水面 tile）
    OpenGround   // 开放地面（等级驱动：0荒 1草 2平原 3肥沃 4丰饶）
}

public enum ResourceType
{
    None,
    Wood,        // 木材 — 森林
    Stone,       // 石材 — 山地/矿山
    Ore,         // 矿石 — 矿山
    Food,        // 粮食 — 开放地面≥3 + 河流
    Gold         // 金币 — NPC交互
}

// ===== ScriptableObject =====
[CreateAssetMenu(menuName = "ValleyRampart/TerrainTileData")]
public class TerrainTileData : ScriptableObject
{
    [Header("地表")]
    public SurfaceType surfaceType;
    public int grade;              // 等级（由 surfaceType 决定范围）
    public TileBase surfaceTile;   // 地表线 tile

    [Header("剖面")]
    public Sprite profileSprite;         // 无河流时的剖面贴图
    public Sprite profileWithRiver;      // 有河流时的剖面变体
    public int profileHeight = 5;        // 剖面深度（tile 数）

    [Header("资源")]
    public ResourceType resourceType;
    public float outputRate;             // 产出速率（由 grade 计算，0=不产出）

    [Header("河流")]
    public bool canHaveRiver = true;     // 是否允许河流流经

    [Header("玩法")]
    public bool canBuild = true;
    public bool isMine;                  // 矿山标记（true → 山地子集）
    public float moveSpeedModifier = 1f; // 沙地 0.8, 雪地等级驱动

    [Header("背景")]
    public Sprite terrainBgSprite;       // 该地形对应的远景图
}
```

## 剖面贴图系统

剖面是**预制大图**，不是逐 tile 拼。每种 TerrainTileData 预画一张完整 Sprite，初始化时直接贴到 TerrainProp Tilemap。

### 贴图规格

| 参数 | 值 |
|------|-----|
| 单 tile 尺寸 | 32×32 px |
| 剖面深度 | 5 tile = 160px |
| 贴图尺寸 | 32×(5×32) = 32×160 px |
| 无河流版（第一期） | 19 张 |
| 河流变体（第二期） | 12 张（需要河流变体的地形） |
| 河流自身 | 1 张（水面+河床） |

## 实现步骤

### 第一期：地表 + 剖面（Day 1-2）

1. 创建 `TerrainTileData.cs` ScriptableObject 脚本
2. 在 `Assets/_Game/Config/Terrain/` 创建 21 个 SO 实例
3. 创建 6 层 Tilemap 场景结构（Grid + 5 个 Tilemap + Decoration 空节点）

5. 编写 `TerrainMapInitializer.cs`：读取配置，在 Surface 层铺地表 tile，在 TerrainProp 层对应位置铺剖面 Sprite
6. 绘制 300×50 的河谷地图：左到右依次经过多段不同地形
7. 主城放在 tile ~100 位置（开放地面等级≥2）

1. 绘制河流变体剖面（12 张）

3. 绘制地形对应背景图，挂到 TerrainBg 层
4. SkyBg 层添加天空渐变 + 云
