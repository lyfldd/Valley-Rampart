# 🎚️ 难度系统设计 v3（数据驱动版）

> 2026-07-23 阿铁拍板，主理人游承峰汇编
> 设计哲学：配置走 SO 数据驱动 + 累计系数走 ISaveable 存档，复用项目已有架构

## 📝 设计哲学

| 数据类型 | 存储方式 | 加载时机 | 示例 |
|---------|---------|---------|------|
| **配置数据**（只读） | ScriptableObject (.asset) | 初始化加载 | 三档预设、基础资源、季节增长量 |
| **运行时累计数据**（可变） | ISaveable + JSON 存档 | 每次加载存档 | 当前难度系数、当前档位 |

两类数据完全分离：SO 是"规则"，存档是"状态"。改规则不动代码，改状态自动持久化。

## 一、核心设计

### 1.1 DifficultyFactor = Difficulty 档位值（自洽）

| 档位 | Difficulty 值 | DifficultyFactor 初始值 |
|------|:--:|:--:|
| Easy | 1 | 1.0 |
| Normal | 2 | 2.0 |
| Hard | 3 | 3.0 |

**自洽点**：最低难度系数=1，玩家选 Easy 时系数=1（最低），Hard 时=3（最高）。

### 1.2 初始资源公式

```
初始资源 = 基础资源 ÷ DifficultyFactor
```

**基础资源**（最低难度下的满额）：200 金/石/木/粮（可在 SO 配置）

| 档位 | DifficultyFactor | 初始资源（每种） |
|------|:--:|:--:|
| Easy | 1 | 200 / 1 = **200** |
| Normal | 2 | 200 / 2 = **100** |
| Hard | 3 | 200 / 3 ≈ **66**（向下取整） |

### 1.3 运行时增长

- 每季 `+0.25`（`TimeManager` 发布 `SeasonChangedEvent` 时触发）
- 每年（4季）`+1.0`
- 无限塔防模式继续每季 `+0.25`，无限增长

### 1.4 totalDays 最终定位

- **不再是玩家开局选择的**
- `TimeManager` 内部从 Day 1 自动累计
- 仅用于触发季节变化、记录游戏进度
- **不参与难度计算、不作为胜利条件**

## 二、数据结构

### 2.1 DifficultyConfig（SO，配置数据）

**文件**：`Assets/_Game/Data/DifficultyConfig.cs`
**资产实例**：`Assets/Resources/Difficulty/DifficultyConfig.asset`

```csharp
using UnityEngine;

/// <summary>
/// 难度系统配置（ScriptableObject，数据驱动）。
/// 存储三档预设 + 基础资源 + 季节增长量。
/// 修改数值不需要改代码，Inspector 直接调。
/// </summary>
[CreateAssetMenu(menuName = "ValleyRampart/DifficultyConfig", fileName = "DifficultyConfig")]
public class DifficultyConfig : ScriptableObject
{
    [Header("三档难度预设（索引 0/1/2 对应 Easy/Normal/Hard）")]
    public DifficultyPreset[] presets = new DifficultyPreset[3];

    [Header("基础资源（最低难度满额，实际初始 = 基础 ÷ 难度系数）")]
    public int baseGold = 200;
    public int baseStone = 200;
    public int baseWood = 200;
    public int baseFood = 200;

    [Header("季节增长")]
    [Tooltip("每季难度系数增加多少")]
    public float factorGrowthPerSeason = 0.25f;

    /// <summary>按档位值获取预设（档位值 1/2/3 → 索引 0/1/2）。</summary>
    public DifficultyPreset GetPreset(int difficultyValue)
    {
        int index = Mathf.Clamp(difficultyValue - 1, 0, presets.Length - 1);
        return presets[index];
    }

    /// <summary>按档位值算初始资源（基础 ÷ 系数）。</summary>
    public ResourcePack GetInitialResources(int difficultyValue)
    {
        return new ResourcePack
        {
            gold = Mathf.FloorToInt(baseGold / difficultyValue),
            stone = Mathf.FloorToInt(baseStone / difficultyValue),
            wood = Mathf.FloorToInt(baseWood / difficultyValue),
            food = Mathf.FloorToInt(baseFood / difficultyValue)
        };
    }
}

/// <summary>单档难度预设。</summary>
[System.Serializable]
public struct DifficultyPreset
{
    public string name;              // "Easy" / "Normal" / "Hard"
    public int difficultyValue;      // 1 / 2 / 3（= DifficultyFactor 初始值）
    [Tooltip("现实秒/天。Easy 节奏慢，Hard 节奏紧")]
    public float secondsPerDay;      // 传给 TimeManager
    [Tooltip("每季天数")]
    public int daysPerSeason;        // 传给 TimeManager
}

/// <summary>资源包（四资源统一结构）。</summary>
[System.Serializable]
public struct ResourcePack
{
    public int gold;
    public int stone;
    public int wood;
    public int food;
}
```

**资产实例默认值**（Inspector 配置）：

| 字段 | Easy (索引0) | Normal (索引1) | Hard (索引2) |
|------|:--:|:--:|:--:|
| name | Easy | Normal | Hard |
| difficultyValue | 1 | 2 | 3 |
| secondsPerDay | 600 | 480 | 360 |
| daysPerSeason | 10 | 10 | 10 |

基础资源：baseGold/Stone/Wood/Food = 200；factorGrowthPerSeason = 0.25

### 2.2 DifficultyManager（Singleton + ISaveable，运行时）

**文件**：`Assets/_Game/Systems/Difficulty/DifficultyManager.cs`

```csharp
using UnityEngine;

/// <summary>
/// 难度系统运行时管理器。
/// - 新建游戏：Initialize(difficulty) 设初始系数
/// - 读档：LoadState 自动恢复累计系数
/// - 运行时：订阅 SeasonChangedEvent，每季 +0.25
/// - 消费方：读 CurrentFactor 调整敌人强度
/// </summary>
public class DifficultyManager : Singleton<DifficultyManager>, ISaveable
{
    public string SaveId => "DifficultyManager";
    public SaveLoadPhase LoadPhase => SaveLoadPhase.Global;

    [Header("配置引用")]
    [Tooltip("Inspector 拖入，留空则从 Resources/Difficulty/ 加载")]
    [SerializeField] private DifficultyConfig config;

    // ===== 运行时状态 =====

    /// <summary>当前难度档位值（1/2/3）。</summary>
    public int CurrentDifficulty { get; private set; }

    /// <summary>当前难度系数（初始=档位值，每季+0.25）。</summary>
    public float CurrentFactor { get; private set; }

    /// <summary>配置引用（供其他系统读取预设参数）。</summary>
    public DifficultyConfig Config => config;

    protected override void Awake()
    {
        base.Awake();

        // 加载配置（Inspector 优先，其次 Resources）
        if (config == null)
        {
            config = Resources.Load<DifficultyConfig>("Difficulty/DifficultyConfig");
        }

        if (config == null)
        {
            Debug.LogError("[DifficultyManager] 未找到 DifficultyConfig！请创建资产放在 Resources/Difficulty/ 下");
        }

        SaveManager.Instance.RegisterSaveable(this);
    }

    private void Start()
    {
        EventBus.Subscribe<SeasonChangedEvent>(OnSeasonChanged);
    }

    private void OnDestroy()
    {
        EventBus.Unsubscribe<SeasonChangedEvent>(OnSeasonChanged);
    }

    // ===== 新建游戏初始化 =====

    /// <summary>新建游戏时由 GameBootstrap 调用。</summary>
    public void Initialize(int difficulty)
    {
        CurrentDifficulty = Mathf.Clamp(difficulty, 1, 3);
        CurrentFactor = CurrentDifficulty;  // 初始系数 = 档位值（自洽）

        // 同步配置到 TimeManager（每天时长、每季天数）
        DifficultyPreset preset = config.GetPreset(CurrentDifficulty);
        TimeManager.Instance.SetSecondsPerDay(preset.secondsPerDay);
        TimeManager.Instance.SetDaysPerSeason(preset.daysPerSeason);

        Debug.Log($"[DifficultyManager] 初始化: 档位={CurrentDifficulty}, 系数={CurrentFactor}, "
            + $"秒/天={preset.secondsPerDay}, 天/季={preset.daysPerSeason}");
    }

    /// <summary>获取当前档位的初始资源（供 RulerController 用）。</summary>
    public ResourcePack GetInitialResources()
    {
        return config.GetInitialResources(CurrentDifficulty);
    }

    // ===== 运行时：每季增长 =====

    private void OnSeasonChanged(SeasonChangedEvent evt)
    {
        float oldFactor = CurrentFactor;
        CurrentFactor += config.factorGrowthPerSeason;

        Debug.Log($"[DifficultyManager] 季节变化 {evt.OldSeason}→{evt.NewSeason}: "
            + $"系数 {oldFactor:0.00} → {CurrentFactor:0.00}");

        EventBus.Publish(new DifficultyChangedEvent(oldFactor, CurrentFactor));
    }

    // ===== ISaveable 实现 =====

    public SavePayload SaveState()
    {
        var data = new DifficultySaveData
        {
            currentDifficulty = CurrentDifficulty,
            currentFactor = CurrentFactor
        };
        return new SavePayload
        {
            typeName = typeof(DifficultySaveData).AssemblyQualifiedName,
            json = JsonUtility.ToJson(data),
            version = 1
        };
    }

    public void LoadState(SavePayload payload)
    {
        if (payload.typeName != typeof(DifficultySaveData).AssemblyQualifiedName)
        {
            Debug.LogWarning("[DifficultyManager] 存档类型不匹配，跳过。");
            return;
        }

        var data = JsonUtility.FromJson<DifficultySaveData>(payload.json);
        CurrentDifficulty = data.currentDifficulty;
        CurrentFactor = data.currentFactor;

        // 读档时也要同步 TimeManager 配置
        DifficultyPreset preset = config.GetPreset(CurrentDifficulty);
        TimeManager.Instance.SetSecondsPerDay(preset.secondsPerDay);
        TimeManager.Instance.SetDaysPerSeason(preset.daysPerSeason);

        Debug.Log($"[DifficultyManager] 读档恢复: 档位={CurrentDifficulty}, 系数={CurrentFactor:0.00}");
    }
}
```

## 三、数据流

### 3.1 新建游戏加载链路

```
玩家在 CharacterCreationPanel 选难度(1/2/3)
  │
  ▼
NewGameConfig { rulerName, difficulty, mapSeed }  ← 删除了 totalDays, selectedSlotId
  │
  ▼
GameBootstrap.InitializeScene()
  │
  ├─ DifficultyManager.Instance.Initialize(config.difficulty)
  │     ├─ CurrentFactor = difficulty（1/2/3）
  │     └─ 同步 TimeManager 的 secondsPerDay / daysPerSeason
  │
  ├─ RulerController 用 DifficultyManager.GetInitialResources() 设初始资源
  │     └─ 初始资源 = baseResources / CurrentFactor
  │
  └─ TimeManager 从 Day 1 开始自动累计（不再需要 SetTotalDays）
```

### 3.2 读档加载链路

```
SaveManager.Load(slotId)
  │
  ├─ 阶段1 Global：DifficultyManager.LoadState(payload)
  │     ├─ 恢复 CurrentDifficulty + CurrentFactor（累计值）
  │     └─ 同步 TimeManager 配置
  │
  ├─ 阶段1 Global：TimeManager.LoadState(payload)
  │     └─ 恢复 CurrentDay + CurrentSeason
  │
  └─ 阶段2 Scene：RulerController.LoadState(payload)
        └─ 恢复资源数值（不用重算，直接读存档）
```

### 3.3 运行时增长链路

```
TimeManager 每过 daysPerSeason 天
  │
  ▼
发布 SeasonChangedEvent
  │
  ▼
DifficultyManager.OnSeasonChanged()
  │
  ├─ CurrentFactor += 0.25
  └─ 发布 DifficultyChangedEvent(oldFactor, newFactor)
        │
        ├─→ WaveManager（未实现）：敌人血量/数量 × newFactor
        ├─→ CombatSystem（未实现）：伤害计算参考 newFactor
        └─→ UI（可选）：显示当前难度系数
```

## 四、和其他系统的结合

### 4.1 存档系统（SaveManager）

**已有架构，无需改动**。DifficultyManager 实现 `ISaveable`：
- `SaveId = "DifficultyManager"`
- `LoadPhase = Global`（全局阶段，和 TimeManager / WorldManager 同阶段，最先恢复）
- `SaveState()` → 序列化 `DifficultySaveData`（currentDifficulty + currentFactor）
- `LoadState()` → 反序列化恢复

SaveManager 自动收集、自动分发，**不需要改 SaveManager 代码**。

### 4.4 RulerController

**改动**：
- `ApplyRulerData()` 方法：不再直接读 RulerData 的 initialGold/Stone/Wood/Food

```csharp
// 改动后的 ApplyRulerData
private void ApplyRulerData()
{
    if (rulerData == null) return;

    // 初始资源改由难度系统算（基础资源 ÷ 难度系数）
    ResourcePack initial = DifficultyManager.Instance.GetInitialResources();
    Gold = initial.gold;
    Stone = initial.stone;
    Wood = initial.wood;
    Food = initial.food;

    Debug.Log($"[RulerController] 难度{DifficultyManager.Instance.CurrentDifficulty} 初始资源: "
        + $"G={Gold}, S={Stone}, W={Wood}, F={Food}");
}
```
