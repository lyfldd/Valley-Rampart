## 🔄 阶段 2：世界骨架（第 1-3 周，Day 3-21）

> 核心原则：**按依赖层级开发，不跳级**。
> L1 → L2 → L3，每一层为上一层提供材料。

### 层级总图

```
L1 地基 ─── 任何系统都依赖它
  ├─ EventBus ✅        系统间通信
  ├─ 君主移动 ✅        玩家操作基础
  ├─ Tilemap ✅         地图基础
  └─ 基础 UI ✅         信息展示

L2 框架 ─── L3 都要用它
  ├─ 时间系统 ⬜        日夜循环、季节、天灾触发
  ├─ 资源管理器 ⬜      木/石/金/粮 四资源采集/消耗/存储
  ├─ 建造系统 ⬜        建造网格 + 虚影预览 + 资源产出建筑
  └─ 君主技能 ⬜        特质选择 + 主动技能冷却

L3 玩法 ─── 核心体验
  ├─ 战斗系统 ⬜        箭塔/投石机/炸弹/陷阱/伏击
  ├─ 敌人波次 ⬜        裂隙刷怪 + 左右路径寻路
  ├─ 动态阵营 ⬜        四段好感 + AI博弈 + 外交建筑
  ├─ 民生系统 ⬜        村民温饱/士气/人口
  ├─ 科技树 ⬜          建筑/兵种/农业/工事四分支
  ├─ 天灾系统 ⬜        寒冬/洪水/血月BOSS
  └─ 跨岛远征 ⬜        坐船去其他岛屿，双路线结局

L4 周边
  ├─ 存档系统 ⬜        单槽位快速存档
  ├─ UI 面板 ⬜         建造菜单/外交面板/波次提示
  └─ 音效/特效 ⬜       简单反馈
```

### 📦 L1 地基层

| 系统 | 负责 | 内容概要 |
|------|------|----------|
| **EventBus** | 全局通信 | 约 15 个 core event struct，直接复用僵尸项目模板 |
| **君主移动** | 玩家控制 | 左右移动 + 骑马动画 + 奔跑/慢走切换 |
| **Tilemap 地图** | 场景基础 | 单 Tilemap + 7 种复合地形大图横向拼接，7 个 TerrainTileData SO |
| **基础 UI** | 信息展示 | 顶部资源条 + 日夜指示器 + 波次倒计时 |

### 📦 L2 框架层

| 系统 | 负责 | 内容概要 |
|------|------|----------|
| **时间系统** | 日夜/季节 | 白天 3-5min → 夜晚 2-3min，季节轮换（春夏秋冬），天灾触发概率 |
| **资源管理器** | 经济基础 | 木/石/金/粮 四资源池，自动采集（伐木场/采石场/农田/码头），消耗追踪 |
| **建造系统** | 防线搭建 | BuildingData(SO) 驱动，GhostPreview 虚影预览，建造进度条，材料验证，资源产出建筑（伐木场/矿洞/农场） |
| **君主技能** | 战术操作 | RulerTrait(SO) 配置，3 主动技能（战吼/紧急维修/骑兵冲锋），白天冷却 |

### 📦 L3 玩法层

| 系统 | 负责 | 内容概要 |
|------|------|----------|
| **战斗系统** | 伤害管线 | IDamageable 接口，箭塔自动瞄准 + 投石机范围伤害 + 炸弹/陷阱/伏击 |
| **敌人波次** | 刷怪逻辑 | WaveData(SO) 配波次，裂隙定点刷新，左右路径寻路，逐层突破防线 |
| **动态阵营** | 外交博弈 | 四段好感（死敌/中立/友好/同盟），进贡/通商/协同清剿，3 外交建筑，AI 互相攻伐 |
| **民生系统** | 村民管理 | 饥饿/士气/人口三值，自动消耗粮食，低士气→人口流失，高士气→采集加速 |
| **科技树** | 成长系统 | 建筑/兵种/农业/工事四分支，每分支 3 级，消耗金币+时间 |
| **天灾系统** | 环境压力 | DisasterData(SO) 配置，寒冬（粮耗×2）、洪水（壕沟失效）、血月（BOSS+死敌联军） |
| **跨岛远征** | 关卡推进 | ExpeditionManager，码头建造→乘船→抵达新岛屿（不同AI势力），双路线结局 |

| `BuildingData` | 建筑属性（成本/耐久/功能/升级链） | ~30 |

| `WaveData` | 波次配置（怪物类型/数量/间隔） | ~20 |
| `EnemyData` | 敌人属性（小怪/BOSS/敌国兵种） | ~15 |
| `RulerTraitData` | 君主特质（筑城者/骑兵统帅/外交官） | ~5 |

## 📅 第三周：Lightweight GDD（快速落地用）

> 每个子系统只写到"能直接开工"的程度：文件数 + 核心字段 + 关键伪代码。

### 建造系统
```
文件：BuildController.cs / BuildingData.cs(SO) / GhostPreview.cs
核心逻辑：
  - 读取 BuildingData.cost → 校验资源 → GhostPreview 显示 → 点击确认 → 扣资源 → SetTile
  - BuildingData 字段：tile, cost(4资源), buildTime, upgradeTo, defenseLine(壕沟/外墙/内墙/主城)
```

### 战斗系统
```
文件：CombatSystem.cs / IDamageable.cs / Projectile.cs
核心逻辑：
  - 箭塔每 N 秒扫描范围内最近的敌人 → 生成箭矢 Projectile → 碰撞后调用 IDamageable.TakeDamage()
  - 投石机范围伤害：落地点半径内所有敌人分摊伤害
  - 炸弹/陷阱/伏击：一次性触发，高伤害
```

### 动态阵营
```
文件：DiplomacyManager.cs / FactionConfigData.cs(SO)
核心逻辑：
  - 好感度 float 0~100 → 四段判定
  - 进贡：扣资源 + 好感瞬间上涨
  - 通商：互市码头每秒缓慢涨好感
  - 协同清剿：摧毁共享裂隙 + 好感
  - AI 好感衰减：不进贡/不通商/见死不救 → 持续下降
```

### 地形改造
```
文件：TerrainModification.cs
核心逻辑：
  - 玩家手持铲子 → 在壕沟区 Tilemap 上点击 → 替换 Tile（泥土→壕沟/水/隘口）
  - 每种地形有 EnemyModifier（减速/扣血/绕路）
```

### 敌人波次
```
文件：WaveManager.cs / EnemyController.cs / WaveData.cs(SO)
核心逻辑：
  - 夜晚触发 → 读取 WaveData → 定时从左右裂隙刷新 Enemy Prefab
  - Enemy 向右/左移动 → 遇到壕沟减速 → 遇到外墙攻击墙体 → 墙体耐久归零 → 继续前进
  - 到达主城 → 扣主城耐久
```
