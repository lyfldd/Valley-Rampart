# 2026-07-09 工作日志

## 1.4 Tilemap 河谷地图 — 全面重构

- 从 3 层 Tilemap 升级为 6 层架构：SkyBg / TerrainBg / Surface / TerrainProp / Water / Decoration
- 确定了 Y 轴三区布局：上区背景地景、中区游戏内容、下区材质剖面
- 剖面层纯视觉，由 TerrainTileData 数据驱动，不可交互
- 地形类型：森林/山地/矿山/沙地/雪地/山壁/河流 + 开放地面（0荒地→4丰饶）
- 矿山是山地子集，同时产石材和矿石
- 开放地面用等级统一管理视觉变化（不用多类型）
- 雪地等级由 SeasonManager 季节驱动
- 河流双重身份：独立 tile + 邻接地形 hasRiver 属性
- 五资源地形来源明确：木材→森林、石材→山地/矿山、矿石→矿山、粮食→开放地面≥3+河流、金币→NPC交互
- 数据模型：TerrainTileData(ScriptableObject)，21 个 SO 实例
- 剖面贴图第一期 19 张（无河流版），留 32×160px 规格
- 主计划书 1.4 描述同步更新
